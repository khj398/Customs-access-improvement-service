물품명 단독으로는 자전거/타이어/산업용 공기펌프와 구분 불가 → **이름 기반 규칙의 본질적 한계**.

```
토큰: [HAND, PUMP]
적용 규칙: industry_pump (priority 40, keywords_any=[PUMP])
실제 의도: sports_balloon_pump (priority 15, keywords_all=[BALLOON])
                                  ↑ BALLOON 토큰이 없어 미적용
```

### 개선 방안

| 방법 | 효과 | 비용 |
|------|------|------|
| 로트 내 공동 분류 (같은 경매의 타 물품명 참조) | 정확 | 구조 변경 필요 |
| `HAND + PUMP` → 완구로 처리 | 부작용 (자전거 펌프 오분류 위험) | — |
| OpenAI 위임 (fallback) | 정확 | API 비용 |
| 현상 유지 (오분류율 2%, 허용 기준 이내) | — | 없음 |

**결정**: 현상 유지. 단일 이름 기반 분류에서 2%는 허용 오차 이내. OpenAI 위임 시 해결 가능.
